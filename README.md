# LoteQA - Sistema de Gestión de Lotes y Control de Calidad para AOVE

## Descripción del Proyecto

Plataforma web para el registro, seguimiento y análisis de lotes de producción de Aceite de Oliva Virgen Extra (AOVE). El objetivo es demostrar habilidades de **pensamiento lógico**, **QA (Quality Assurance)**, **análisis de datos** y prácticas básicas de **DevOps**.

## Objetivos de Aprendizaje

- Desarrollar pensamiento lógico y capacidad de reporte claro de problemas
- Diseñar y trabajar con bases de datos reales
- Construir backend y frontend funcional
- Aplicar prácticas profesionales de control de versiones, testing y documentación
- Crear un proyecto de portafolio orientado a QA + DevOps + Data Analysis

## Estado Actual
